# MonteCarloMetropolisKT
## To install
make executable 

```bash
python3 -m venv venv
```
activate the virtual environment:
```bash
source venv/bin/activate
```

For Windows:
```bash
venv\Scripts\activate
```

Then install the requirements:
```bash
pip install -r requirements.txt
```

## Access the Jupyter Notebook
```bash
jupyter notebook
```
